Different types of input styles

When it comes to designing a form in HTML, input tags play a crucial role in capturing user input. However, styling them effectively can enhance the form's overall look and feel, making it more attractive and user-friendly. In this blog post, we'll explore three simple and visually appealing designs for input tags that you can easily implement in your HTML forms.

The first design is a flat-style input tag that uses simple outline with curved border and color variations to highlight the focused input field. This design is clean and minimal, making it perfect for modern and professional websites.

    <style>
      .input {
        width: fit-content;
        height: fit-content;
        position: relative;
      }
      .inputf {
        width: 300px;
        height: 35px;
        font-family: poppins;
        border: none;
        outline: 0.3mm solid rgb(118, 118, 118);
        box-sizing: border-box;
        padding: 0 10px;
        border-radius: 1mm;
        position: relative;
        z-index: 1;
      }
      .inputf:focus {
        outline: 0.7mm solid rgb(19, 116, 214);
      }
      .label {
        font-family: poppins;
        font-size: 12px;
        font-weight: 500;
        position: absolute;
        top: 0;
        left: 0;
        z-index: 0;
        transition: 0.25s;
        color: rgb(19, 116, 214);
      }
      .inputf:focus + .label {
        top: -20px;
      }
      .inputf:not(:placeholder-shown) + .label {
        top: -20px;
      }
      .inputf:not(:placeholder-shown) {
        outline: 0.7mm solid rgb(19, 116, 214);
      }
    </style>
    <div class="input">
      <input type="text" name="" id="" class="inputf" placeholder="Name" />
      <span class="label">Name</span>
    </div>
